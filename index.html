Sql Server

SQL Server is a relational database management system (RDBMS) developed by Microsoft. It is widely used to store, organize and manage data in applications of different types, especially in corporate environments. SQL Server uses the T-SQL (Transact-SQL) language, an extension to standard SQL, which adds additional functionality such as transactions, error handling and flow controls.

Neo4J

Neo4j is a graph-oriented database management system, designed to store, manipulate and query data structured as graphs. Unlike relational databases that use tables and primary/foreign keys, Neo4j stores data in nodes, relationships and properties, which makes it especially suitable for applications involving complex networks such as social networks, recommendations, fraud management and more.
